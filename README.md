**Recommended software versions:**
----

* MongoDB >= 4.0.5

* node.js >= 10.11.0

* sails.js >= 1.1.0

* mocha >= 6.0.2 (for running tests)


**Usage**
----

1. git clone
2. Create file "local.js" in folder "config" and fill it in by example "stuff/local_example.js"
3. `npm install`
4. `sails lift`

To run tests:  
`npm test`
